Questions fréquentes

Le chalet accepte-t-il les animaux ?

Les animaux ne sont pas acceptés afin de préserver le confort de tous les voyageurs.

Peut-on stationner facilement ?

Oui, le stationnement est possible devant le chalet. Le terrain n’est pas encore clôturé.

Le linge est-il inclus ?

Le linge de maison et les serviettes sont proposés en option. Vous pouvez aussi apporter les vôtres.

Comment se passe l’arrivée ?

L’arrivée autonome est possible grâce à une boîte à clés. Les instructions sont envoyées avant le séjour.
